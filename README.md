# AVERX 官方网站（Open Source Project）

欢迎来到 **AVERX** 的官方网站开源仓库！  
这是一个面向全球开发者开放的互联网品牌项目，任何人都可以查看、Fork、提交代码和参与建设。

🌐 在线访问网站：  
👉 https://aexar-vv.github.io/averx-site/

---

## 🚀 项目简介

AVERX 是一个专注于 **个人品牌、科技风格、数字化呈现** 的互联网视觉项目。  
目标是打造一个：

- 极简  
- 科技  
- 高级感  
- 可不断迭代  

的现代品牌网站。

网站目前基于 **纯前端结构（HTML + CSS + JS）**，适合所有开发者参与贡献。

---

## 🌍 面向全球开发者开放（欢迎 Fork / PR）

我们欢迎各种类型的贡献：

### 🟦 可以参与的方向
- 改进首页 UI（科技风 / 动效）
- 重构前端结构（原生 / React / Vue / Astro）
- 添加动效（GSAP、粒子背景）
- 多语言支持（中 / 英）
- 商店页面优化
- 新增模块：博客 / 联系方式 / 品牌故事
- 美化布局，提升体验

欢迎自由发挥，只要保持 AVERX 的“未来科技风”即可。
